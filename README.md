# Krokomierz-GPS
Aplikacja mobilna stworzona w środowisku MIT App Inventor umożliwiająca śledzenie lokalizacji GPS urządzenia oraz pomiar liczby kroków pomiędzy sekcjami Start i Stop.
System zapisuje dane do lokalnej bazy danych TinyDB i umożliwia przeglądanie dziennika zapisów.

## Wymagania sprzętowe
Telefon, tablet lub emulator:
- System operacyjny Android w wersji 2.1 lub nowszej,
- Działający sensor GPS i akcelerometr,
- Przynajmniej 30MB przestrzeni dyskowej,
- Poprawnie skonfigurowana data i godzina w systemie.

## Instrukcja instalacji
1. Zezwól urządzeniu na pobieranie i instalację aplikacji z nieznanych źródeł,
2. Pobierz plik APK,
3. Uruchom plik i zainstaluj aplikację,
4. Uruchom aplikację,
5. Zezwól aplikacji na dostęp do lokalizacji,

## Instrukcja użytkowania
W celu rozpoczęcia pomiaru kroków uruchom aplikację, zezwól na dostęp do lokalizacji urządzenia
oraz kliknij zielony przycisk "Start".

Aby wstrzymać licznik kroków kliknij czerwony przycisk "Stop", który znajduje się w miejscu
poprzednio klikniętego przycisku "Start".

Aby zapisać dane kliknij czerwony przycisk "Resetuj i zapisz". Następnie możesz przejść do dziennika, aby zobaczyć nowy wpis.
